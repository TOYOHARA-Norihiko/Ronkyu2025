# 経済学論究；関西地域における景気先行性の時間–周波数分析（2025　豊原）
## ― ウェーブレット・コヒーレンシーおよびコヒージョンによる検証 ―

このリポジトリは、同タイトルの論文におけるデータ分析および図表作成に使用した R スクリプト（R Markdown形式）を公開するものです。

## 概要
本研究では、関西地域における景気先行指標（CLI）と一致指標（CI_C）の関係を、ウェーブレット分析を用いて時間・周波数の両面から分析しています。
- **手法**: 連続ウェーブレット変換、ウェーブレット・コヒーレンシー、コヒージョン分析
- **主要なツール**: R (`WaveletComp`, `plotly`, `openxlsx`)

## ファイル構成
各 `.Rmd` ファイルは論文内の図表に対応しています。

- `fig1.Rmd`: Morletウェーブレット基底関数の視覚化
- `fig2.Rmd`: 3Dウェーブレット変換（時間・周期・振幅）の描画
- `fig3.Rmd`: 各府県のウェーブレット・パワースペクトラム
- `fig4.Rmd`: CLIとCI_Cのコヒーレンシーおよび位相差分析
- `fig5.Rmd`: ラグ・シフトによる先行性の頑健性チェック
- `fig6.Rmd`: 関西6府県の同調性（コヒージョン）分析

## 実行方法

### 1. データの準備
実行には、以下のデータファイルがスクリプトと同一のディレクトリに必要です。あらかじめダウンロードのこと。
- `kansai_cli_ci_c.xlsx`
※A列に日付、B列以降に2018年1月から2025年8月までの各指標（`osaka_cli`, `osaka_ci_c` 等）が格納されています。

### 2. 必要パッケージのインストール
R上で以下のコマンドを実行して、必要なパッケージをインストールしてください。
install.packages(c("openxlsx", "WaveletComp", "plotly", "dplyr", "gifski", "htmlwidgets"))
